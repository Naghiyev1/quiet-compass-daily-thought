# Quiet Compass v2.0 — 365 Thoughts + Wallpaper Styles

## What changed

- 365 daily thoughts, enough for a full year
- Better wallpaper typography spacing
- Four visual styles:
  - Calm
  - Dark
  - Paper
  - Sunrise
- No weekly theme layer
- Favourite thoughts
- Archive
- 1080×1920 PNG wallpaper export

## Copyright approach

The app uses original reflections inspired by serious books and traditions. It avoids long copyrighted quotes from modern books.
